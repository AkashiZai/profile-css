จากการปฏิบัติงานใน Lab นี้ ทำให้ผมเข้าใจถึงความสำคัญของการใช้ Desing css เพื่อตกเเต่งหน้าเว็ปไซต์ให้ได้อย่างถูกต้องและอ่านเข้าใจง่าย นอกจากนี้ยังได้พัฒนาทักษะการเลือกใช้แท็กพื้นฐานต่างๆ ให้เหมาะสม ทั้งการจัดลำดับ div ในการสร้าง class ในการใช้ CSS, การออกเเบบ container ของเเต่ละส่วนของ CSS รวมถึงการออกเเบบฟอร์มรับข้อมูล ซึ่งถือเป็นพื้นฐานสำคัญในการพัฒนาเว็บที่มีคุณภาพและรองรับการใช้งานจริงครับ เเละสามารถเอาวิธีเขียนของ CSS ไปต่อยอดในการเขียนภาษาอื่นๆได้ โดยการจัดเรียงโค้ดให้สวยงามง่ายต่อการอ่าน ทำให้การเขียนโค้ดเป็นระบบมากขึ้น
